Quick Start¶

  1. Install CRUX (link). For detailed instructions, see Installing CRUX section

  2. If using your own data: Use the Data => Import Data module to load a MAF.

  3. Select Single Cohort Statistics and choose a dataset to analyse.

  4. Scroll down to view visualisations.

  5. Repeat for any module on the sidebar.
